Scraping Mars

Web App to Scrape Various Sites for Mars Data

This web application scrapes NASA Mars News, JPL Mars Images, Space Facts, USGS Astrogeology Science Center, and the Mars Weather Twitter account to get data for the Mission to Mars and displays it in a single html page. Clicking the Scrape New Data will do exactly that and update the page. I've attached a screenshot below to show what it looks like. All of my code is here.
